# Usage

This guide covers the basic workflow for running live sales with Comment Boost.

## Running a Live Sale

### Before You Start

1. Ensure all products are added and have sufficient stock
2. Test your Facebook connection
3. Have your pricing and discounts ready

### During the Live Stream

1. **Start the Sale Session**
   - Go to **Dashboard** > **Sales**
   - Click **Start Live Sale**
   - This activates real-time order processing

2. **Process Orders**
   - Orders appear automatically as customers comment
   - Review and confirm each order
   - Stock is automatically updated

3. **End the Session**
   - Click **End Sale** when finished
   - Review the session summary
   - Process pending orders

## Order Management

### Order Statuses

| Status | Description |
|--------|-------------|
| Pending | Order received, awaiting confirmation |
| Confirmed | Order confirmed, ready for processing |
| Shipped | Order has been shipped |
| Delivered | Order delivered to customer |
| Cancelled | Order was cancelled |

### Processing Orders

1. Go to **Dashboard** > **Orders**
2. Filter by status if needed
3. Click an order to view details
4. Update status as order progresses

## Tips for Success

::callout{type="tip"}
**Pro Tip:** Use product codes during live streams for faster order processing. Customers can comment the code to order.
::

- Keep your product descriptions clear and concise
- Update stock levels in real-time
- Respond to customer questions promptly
- Review analytics after each session to improve
